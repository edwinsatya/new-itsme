# Media slots

Drop real files here with these exact names and rebuild — the site swaps them
in automatically. Until a file exists, its slot renders a designed fallback
(navy space, acid ring, ultramarine bloom) instead of a broken image.

| File                    | Used in                                   |
| ----------------------- | ----------------------------------------- |
| `hero-loop.mp4`         | Hero background loop (muted, looped)      |
| `portal.jpg`            | Expanding portal frame                    |
| `portrait.jpg`          | Intro fold floating portrait              |
| `proj-weathernime.jpg`  | Featured panel 01                         |
| `proj-happyfarm.jpg`    | Featured panel 02 + manifesto collage     |
| `proj-magloft.jpg`      | Featured panel 03 + manifesto collage     |
| `proj-bountie.jpg`      | Featured panel 04 (currently designed art)|

Recommended: JPGs ≥ 2000px wide, hero video ≤ 8 MB H.264.
